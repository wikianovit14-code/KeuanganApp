🔥 Hubungkan ke Firebase

Agar data HP & PC sinkron otomatis, masukkan konfigurasi Firebase Anda

1
Buka console.firebase.google.com → Login dengan akun Google → Klik "Add project" → Beri nama (misal: keuangan-wiki)
2
Di halaman project → Klik ikon "</>" (Web) → Register app dengan nama bebas → Copy firebaseConfig
3
Di menu kiri → Build → Firestore Database → Create database → Pilih "Start in test mode" → Done
4
